Dalpha AI 사용 가이드

1. 기본 사용법
Dalpha AI는 인공지능 기반 데이터 분석 도구입니다. 누구나 쉽게 데이터를 분석할 수 있도록 설계되었습니다.

2. 주요 기능
- 데이터 업로드: CSV, Excel 파일을 직접 업로드할 수 있습니다
- 데이터 분석: 자동으로 데이터를 분석하여 인사이트를 제공합니다
- 시각화: 다양한 차트와 그래프로 데이터를 시각화합니다
- AI 기반 예측: 머신러닝을 통해 미래 트렌드를 예측합니다

3. 시작하기
- 회원가입: www.dalpha.ai에서 무료로 가입할 수 있습니다
- 데이터 준비: 분석하고 싶은 데이터를 준비합니다
- 프로젝트 생성: 새 프로젝트를 만들고 데이터를 업로드합니다

4. 고급 기능
- 커스텀 분석: 원하는 방식으로 데이터를 분석할 수 있습니다
- API 연동: REST API를 통해 외부 서비스와 연동할 수 있습니다
- 협업: 팀원들과 프로젝트를 공유하고 협업할 수 있습니다

5. 요금제
- 무료 체험판: 기본 기능을 무료로 사용할 수 있습니다
- 프로 요금제: 고급 기능과 대용량 데이터 처리가 가능합니다
- 기업용: 맞춤형 솔루션을 제공합니다
